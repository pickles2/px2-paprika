Log

Paprika Framework のコアオブジェクトの1つ $log のオブジェクトクラスです。 このオブジェクトは、Paprika Framework の初期化処理の中で自動的に生成され、$paprika の内部に格納されます。

メソッド $paprika->bowl() を通じてアクセスします。
